Biography

Djefferson Saintilus is a Haitian cybersecurity professional specializing in penetration testing and secure backend development. Known as “Kal” in online communities1, he is recognized for practical problem-solving in web security, combining professional pentesting, open-source tooling, and community education.

Education

  • Completed the Junior Penetration Tester learning path (TryHackMe)2
  • Completed Offensive Penetration Testing coursework (Cybrary)2
  • Certifications: CEH v9 (Certified Ethical Hacker) and LPIC-1 (Linux Professional)3
  • Ongoing Computer Science studies with emphasis on Python and PHP4

Career

  • Secure Society — Co-founder and lead, delivering vulnerability assessments and penetration tests5
  • KWENPAM — Web Penetration Tester (e-commerce, Haiti)4
  • MaketPAMHT — Web Application Security Analyst (e-commerce, Haiti)4
